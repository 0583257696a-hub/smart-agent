# חבילת קודקס — מרכז תפעול לסוכן ביטוח

## מה יש כאן
- docs: מסמכי SPEC מלאים לקודקס
- data: כל המידע במבנה JSON
- source_files: קבצי המקור שהועלו/נמצאו בתיקייה
- index.html / style.css / script.js: אבטיפוס עובד בסיסי

## איך להשתמש
1. פתח את התיקייה ב-VS Code.
2. הפעל Live Server על index.html.
3. בדוק שהמערכת מציגה כרטיסים וחיפוש.
4. תן לקודקס את כל התיקייה הזאת ותבקש ממנו לשפר את המערכת לפי docs.

## הנחיה לקודקס
קרא קודם את:
docs/01_SYSTEM_SPEC.md
ואחר כך את שאר קבצי ה-SPEC.

יש לשמור על UI כמו בתמונה: Sidebar כהה, חיפוש גדול, כרטיסים נקיים, RTL מלא, פרופורציות טובות.
